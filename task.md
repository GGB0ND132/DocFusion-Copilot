## 四、验收口径

- 准确率目标：模板回填准确率 >= 80%。
- 时延目标：单模板回填 <= 90 秒。
- 追溯目标：填充值可追溯到来源文档与证据片段。

## 五、执行命令速查

后端启动：

```bash
cd backend
uvicorn app.main:app --reload --port 8000
```

前端启动：

```bash
cd frontend
npm run dev
```

后端测试：

```bash
cd backend
python -m pytest tests -v
```

测试集结果生成：

```bash
cd backend
python generate_testset_by_requirements.py
```
